# Unichem Distributor System

Sistem informasi berbasis web untuk manajemen pemesanan produk pada sebuah perusahaan manufaktur garam konsumsi dan industri di Indonesia. Dibangun sebagai solusi digitalisasi proses pemesanan distributor yang sebelumnya dilakukan secara manual melalui telepon.

---

## Tentang Project

Perusahaan ini memiliki jaringan distributor yang tersebar di berbagai wilayah Indonesia. Sebelum sistem ini dibuat, proses pemesanan produk oleh distributor dilakukan sepenuhnya melalui telepon — rentan terhadap kesalahan pencatatan dan tidak efisien.

Sistem ini hadir sebagai solusi digital yang memungkinkan distributor melakukan pemesanan langsung melalui website secara mandiri, serta memudahkan admin dalam memantau dan mengelola seluruh transaksi.

---

## Fitur Utama

### Sisi Distributor (Customer)
- Register & Login dengan sistem aktivasi akun
- Dashboard katalog produk dengan fitur pencarian
- Keranjang belanja dan proses Checkout
- Pilihan pembayaran melalui transfer bank
- Riwayat pesanan, tagihan, dan invoice

### Sisi Admin
- Dashboard monitoring pesanan masuk secara real-time
- Grafik ringkasan status pesanan
- Manajemen produk (tambah, edit, hapus, filter status)
- Manajemen invoice dan konfirmasi pembayaran
- Manajemen data distributor dan petugas
- Laporan penjualan lengkap
